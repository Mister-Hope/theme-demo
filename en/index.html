vuepress-theme-hope

A vuepress theme with tons of features✨

Custom theme color and night mode

Supports custom theme colors and allows users to switch between preset theme colors

Night mode

Switch between offical style and a cool night mode theme

Pageviews and comments

with Valine to start pageview statistics and comment support

Enhancements to the default theme

Enhances a range of features based on the default theme, path navigation, footer support, author display, etc.

Icon Support

Icon support through the whole theme

# vuepress-theme-hope is a light vuepress theme

# Install

npm i -D vuepress-theme-hope
1

# Usage

// .vuepress/config.js
module.exports = {
  theme: "hope",
};
